# Contributing

## Adding a Hook or Skill

Each hook or skill lives in its own folder under the relevant tool directory (e.g., `claude-code/hooks/my-hook/`).

### Required files

Every hook/skill folder must contain:

- `README.md` — documentation (see template below)
- The script(s) or configuration files

### README template

```markdown
# Hook/Skill Name

One-sentence description.

## What it does

...

## Requirements

- ...

## Installation

...

## Configuration

...
```

### Folder naming

Use `kebab-case` for folder names.

### Submitting

Open a pull request against `main`. Include a brief description of what the hook/skill does and why it's useful.
