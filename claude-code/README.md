# Claude Code

Hooks and skills for [Claude Code](https://claude.ai/code), Anthropic's CLI for Claude.

## Hooks

Hooks are shell commands that run automatically in response to Claude Code events. See the [hooks README](./hooks/README.md) for how they work.

| Name                                                                  | Trigger | Description                                |
| --------------------------------------------------------------------- | ------- | ------------------------------------------ |
| [long-task-sound-notification](./hooks/long-task-sound-notification/) | `Stop`  | Plays a sound when a task takes ≥5 minutes |
